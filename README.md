# 🤖 Bodhini-AI

> **An Intelligent AI-Powered Career Guidance & Placement Preparation Platform**

Bodhini AI is a comprehensive **AI-powered Career Guidance and Placement Preparation Platform** designed to help students become industry-ready through personalized insights, intelligent analysis, and interactive learning tools.

Built using **Python, Flask, SQLAlchemy, SQLite, HTML, CSS, JavaScript, Jinja2, and Groq LLM**, Bodhini AI combines modern AI capabilities with an intuitive user experience to provide end-to-end career development support.

The platform acts as a **personal AI Career Coach**, helping students improve their resumes, prepare for interviews, analyze skill gaps, predict placement readiness, and generate personalized career roadmaps—all from a single dashboard.
